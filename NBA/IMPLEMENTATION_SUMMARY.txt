API INTEGRATIONS
================================================================================

1. NBA.com Advanced Stats API
   URL: https://stats.nba.com/stats/leaguedashteamstats
   Required Headers:
     - User-Agent: Mozilla/5.0
     - x-nba-stats-origin: stats
     - x-nba-stats-token: true
   Data: TEAM_ABB, ORTG, DRTG, PACE, NET_RATING

2. ESPN Sports API
   URL: https://site.api.espn.com/apis/site/v2/sports/basketball/nba/scoreboard
   Query: dates=YYYYMMDD
   Data: Schedule, scores, game times, team names

3. The Odds API
   URL: https://api.the-odds-api.com/v4/sports/basketball_nba/odds
   Sport: basketball_nba
   Markets: moneyline, spreads, totals
   Regions: us
   Format: american odds

4. Anthropic Claude API (Optional)
   Model: claude-opus-4-6
   Use: Performance feedback and analysis
